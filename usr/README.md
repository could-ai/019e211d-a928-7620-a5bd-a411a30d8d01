# MAMA CARE CAMEROON

MAMA CARE CAMEROON is a digital maternal and child health follow-up system designed for healthcare workers, pregnant women, mothers, and hospitals in Cameroon.

## Core Problem
In Cameroon, weak postnatal follow-up systems lead to missed maternal and child health visits, increasing preventable morbidity and mortality. This platform digitally tracks mothers and children from pregnancy through early childhood and enables healthcare workers to monitor patient records in real time.

## Features
- **Authentication**: Secure Login with modern dark theme UI.
- **Dashboard**: Health summary cards, upcoming appointments, pregnancy progress tracker, and risk alerts.
- **Mother & Child Management**: Register and monitor patients easily.
- **Risk Prediction**: AI-powered insights to flag emergency cases.
- **Responsive UI**: Cross-platform support (Android, iOS, Web) with a professional, minimalist, neon blue accented design.

## Tech Stack
- **Frontend**: Flutter (Dart)
- **Architecture**: Cross-platform (Mobile-first, Web, Desktop)
- **UI/UX**: Dark mode interface with neon blue accents and soft glowing effects.

## Setup & Run
1. Ensure Flutter is installed.
2. Run `flutter pub get`
3. Run `flutter run`
